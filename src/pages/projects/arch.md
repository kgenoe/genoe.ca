---
title: "Arch - API Response Caching Helper"
---

<!-- ![arch icon](./images/arch-icon.png) -->

Arch is simple framework that allows you to use real API data to power your iOS tests. Instead of manually mocking API response data in tests, Arch hits your API once, and caches the response to be used in future runs of your tests.

[![github link](./images/github.png)](https://github.com/kgenoe/arch)
